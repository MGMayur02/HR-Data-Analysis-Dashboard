# HR-Data-Analysis-Dashboard
This project presents an HR Analytics Dashboard built using Power BI to analyze employee demographics, job satisfaction, and attrition trends. The goal of the dashboard is to help HR teams and management understand workforce patterns, identify attrition risks, and support data-driven decision-making.
The dashboard offers a clear and interactive view of key HR metrics, including employee count, attrition rate, age distribution, educational background, and department-wise attrition rates.
